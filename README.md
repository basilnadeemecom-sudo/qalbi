# Qalbi (قَلْبِي) — My Heart

Authentic Islamic duas matched to your emotional state, paired with Names of Allah.

## Deploy on Netlify

1. Drag this folder onto app.netlify.com
2. Change site name to "qalbi" in Site Settings
3. (Optional) Add ANTHROPIC_API_KEY as environment variable for AI-powered matching

## Without API Key
Works perfectly with local keyword matching (no cost, instant results).

## With API Key  
Add `ANTHROPIC_API_KEY` in Netlify → Site Settings → Environment Variables for deep personalised matching.
